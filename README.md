## How to Run

### Prerequisites

* Python 3.x
* Jupyter Notebook / VS Code
* PostgreSQL, MySQL, or SQL Server
* Power BI Desktop

### Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/data-analytics-project.git
```

2. Install required Python packages:

```bash
pip install -r requirements.txt
```

3. Open and run the Python notebooks/scripts.

4. Import the cleaned dataset into your SQL database.

5. Execute SQL queries from the `sql_queries` folder.

6. Open the Power BI dashboard file (`.pbix`) to explore insights.
